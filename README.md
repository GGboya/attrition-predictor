# 离职预测（Attrition Predictor）

本仓库用于基于问卷与行为数据的新员工离职意向 / 离职行为预测相关分析与建模。

## 数据集

| 项目 | 说明 |
|------|------|
| **主数据文件** | `data/处理之后的离职数据-5000.xlsx` |
| **样本量** | 约 5000 条（文件名所示） |
| **格式** | Excel（`.xlsx`） |

同目录下另有原始/辅助表（如 `离职数据-5771.xlsx`）与文献、变量说明 Word 等，请统一放在 **`data/`**，避免与代码混在仓库根目录。

## 变量说明

字段含义与取值编码见 **`docs/variable-labels.md`**（与 `data/变量赋值说明.docx` 等内容一致时可互为参照）。

简要区分：

- **自变量**：性别、高校类型、专业类型、家庭所在地、收入水平、工作匹配度（专业 / 人职）、工作单位性质、工作区域、工作满意度（多维度）、工作机会（多维度）、工作氛围、工作压力等。
- **因变量**：**离职意向**（1–5 分）、**离职行为**（0 = 未离职，1 = 有离职）。

## 仓库结构（初始）

```
attrition-predictor/
├── README.md
├── docs/
│   └── variable-labels.md                   # 变量与取值说明
├── data/                                    # 数据与相关材料（英文目录名）
│   ├── 处理之后的离职数据-5000.xlsx         # 主数据集
│   ├── 变量赋值说明.docx
│   └── …                                    # 其他 xlsx / pdf / docx
└── src/                                     # 后续代码（占位）
```

后续分析脚本、Notebook 等可放在 `src/`、`notebooks/` 等目录中扩展。
