| 取值 | 说明 |

| 2 | 不匹配 |
| 3 | 不确定 |
| 4 | 匹配 |
| 5 | 非常匹配 |

- **专业匹配度**：专业与岗位匹配程度  
- **人职匹配度**：个人与职业 / 岗位整体匹配程度
