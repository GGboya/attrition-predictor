# 数据与预处理

## 3.1 数据来源与样本规模

本研究使用一份来自全国范围应届毕业生的离职调研问卷。原始数据包含 5 771 份记录，覆盖 41 个字段（人口学、学历背景、就业环境、工作评价、离职意向与行为）。经过去重（主键冲突 8 行）、缺失与离群点剔除（具体规则见下）后，最终保留 **N = 5 469** 条有效样本（Table 1）。

- **目标变量**：
  - 离职意向（Y\_INT）— 1~5 级 Likert 有序变量
  - 离职行为（Y\_BH）— 二值（0 = 留任，1 = 已离职）
- **基础率**：整体样本离职率 = **14.5%**，呈显著类别不平衡。
- **切分策略**：Stratified 80/20 切分，冻结 `train_idx.npy`（N_train = 4 375）与 `test_idx.npy`（N_test = 1 094）。所有后续实验共享同一切分以保证可比性。

## 3.2 特征选择

从原始 41 字段中选取 **12 个核心特征**（Table 2），分组为：

| 分组 | 变量 | 可干预 | 类型 |
|---|---|---|---|
| **人口学**（不可变） | 性别、专业类型、家庭所在地 | 否 | 二值/三值 |
| **学历**（不可变） | 高校类型（985/211、普通本科、专科） | 否 | 三值 |
| **工作环境**（不可变） | 工作单位性质、工作区域 | 否 | 多值 |
| **工作评价**（可干预，单调↑） | 工作满意度、工作匹配度、工作机会、工作氛围 | 是 | Likert 1–5 |
| **工作压力**（可干预，单调↓） | 工作压力 | 是 | Likert 1–3 |
| **收入**（可干预，单调↑） | 收入水平 | 是 | 连续（元/月） |

此 12 维特征空间同时用于树基模型（原始编码）与 SVM / LR / kNN（Z-score 标准化）。

## 3.3 预处理流程

1. **整合**：将原始 Excel 多表按主键对齐，统一为 `data/processed/clean.csv`。
2. **缺失处理**：关键字段（工作满意度、工作匹配度、离职行为）缺失的行直接剔除（共 187 行）；对非关键字段按类别众数或分组中位数填补。
3. **编码对齐**：所有分类变量保持整数编码；Likert 变量以原始 1–5 或 1–3 整数形态保留；收入水平保留为元/月单位的浮点数。
4. **标签噪声过滤（Cleanlab v6）**：在训练集上运行 Cleanlab 的 5-fold Confident Learning，识别出 **299 条高度可疑样本**；为它们赋 `weight = 0.3`、其余为 `weight = 1.0`，形成 `sample_weights_v6.npy`。此权重只作用于训练阶段，测试集不做任何权重调整。
