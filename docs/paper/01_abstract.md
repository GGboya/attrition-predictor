# 摘要

**研究背景**　应届毕业生入职后的早期离职给用人单位带来显著的招聘、培训与组织成本。离职预测研究多依赖单一分类器并以整体准确率为目标，既对少数正类（离职）敏感度不足，也难以回答 HR 管理者"应采取何种干预"的决策问题。

**方法**　基于 5 469 份应届毕业生问卷（正类率 14.5%），我们先在 12 个原始变量上构造多基学习器——Random Forest、NR-Boost（基于 XGBoost）、多任务 MLP（同时预测 5 级离职意向与二元离职行为）、RBF 核 SVM 与 Extra Trees——再用 L2-正则 Logistic Regression 作为元学习器进行堆叠，并以 Isotonic 回归校准概率。训练阶段引入 Cleanlab v6 二值权重降低标签噪声的影响。在决策阶段，我们提出 HRCF（Hard-constrained Counterfactual）：通过投影式梯度下降在每一步强制满足不可变、单调与离散取值约束，为高风险员工生成合规、低成本的干预菜单。

**结果**　在独立测试集（N = 1 094）上，冠军集成取得 AUC = 0.784（95% CI [0.750, 0.816]）、Balanced Accuracy = 0.714、在 Bal-Acc 最优阈值 τ = 0.135 下 Sensitivity = 0.730 / Specificity = 0.698，校准 ECE = 0.032。与同一切分上重训的七个基线（LR、RF、XGBoost、LightGBM、CatBoost、kNN、SVM）做 DeLong 配对比较，冠军在 5 个基线上达到显著优势（p < 0.05），与独立调参 RF 差异不显著。SHAP 分析显示工作单位性质、高校类型、收入水平、工作满意度、工作匹配度为预测离职的前五位特征，且与元权重加权的树基模型重要性高度一致（Spearman ρ = 0.888）。HRCF 在 52 名高风险员工上生成的 105 条干预方案的可执行性达 100%（soft-CF 仅为 0%），平均成本 ¥13.9k、平均概率降幅 Δp = 0.309。

**结论**　以"噪声稳健的堆叠预测 + 硬约束反事实干预"为核心的系统在校准精度与可操作性上均优于主流基线，为新员工早期离职的识别与组织干预提供了可部署的闭环框架。

**关键词**　员工离职预测；Stacking 集成；反事实解释；Cleanlab；Isotonic 校准；HR 决策支持
