# 消融实验

## 9.1 堆叠基学习器数量（Table 13）

Phase 6 冠军使用 5 个基学习器；我们消融检验若干变体：

| 变体 | 基组合 | CV OOF AUC | Test AUC | Bal-Acc | 结论 |
|---|---|---|---|---|---|
| **A: Phase 6 CL (5 bases)** | RF + NR-Boost + MT-MLP + SVM + ET | 0.7701 | **0.7844** | **0.714** | **冠军** |
| B: + TabPFN | A + TabPFN | 0.7676 | 0.7802 | 0.701 | TabPFN 权重夺走 0.60，反降整体 AUC |
| C: + gcForest | A + gcForest | 0.7703 | 0.7841 | 0.712 | gcForest 几乎无贡献 |
| D: + kNN | A + kNN | 0.7702 | 0.7835 | 0.710 | kNN 在元层被抑制（coef = 0.02） |
| E: all 8 | A + TabPFN + gcForest + kNN | 0.7685 | 0.7817 | 0.705 | TabPFN 再次抢走权重 |

**结论**：5 基是 AUC / Bal-Acc / 校准三者的共同最优点；加入 TabPFN 会让元学习器"过度信任"TabPFN 预测从而降低整体鲁棒性。

## 9.2 Cleanlab 权重策略（Table 11）

| CL 策略 | 训练样本数 | Test AUC | Test Bal-Acc | 说明 |
|---|---|---|---|---|
| v0: 无 CL | 4 375 | 0.7769 | 0.706 | 基线（无样本权重） |
| v3: 剔除 299 行 | 4 076 | 0.7735 | 0.702 | 丢弃导致方差上升 |
| v5: 连续权重 w ∈ [0, 1] | 4 375 | 0.7798 | 0.708 | 需校准 w 曲线 |
| **v6: 二值权重 (1.0 / 0.3)** | 4 375 | **0.7844** | **0.714** | **最优** |

二值 v6 方案在不牺牲训练集大小的前提下取得最优指标；它的简单性（只有一个"是否可疑"的标识）也便于工程部署。

## 9.3 概率校准方法（Table 16）

| 方法 | Test AUC | Test ECE | Test Brier |
|---|---|---|---|
| 未校准（L2-LR 原输出） | 0.7844 | 0.0894 | 0.1246 |
| **Isotonic (5-fold CV OOF fit)** | 0.7844 | **0.0321** | **0.1100** |
| Platt (sigmoid) | 0.7844 | 0.0452 | 0.1135 |

校准不改变 AUC（后处理为单调变换）但大幅提升校准质量。Isotonic 优于 Platt 的原因是 stack meta 概率分布偏右倾，不适合 sigmoid 强假设。

## 9.5 元学习器选择

| 元学习器 | CV OOF AUC | Test AUC |
|---|---|---|
| **L2-LR (C=10)** | **0.7701** | **0.7844** |
| Logistic (balanced class weight) | 0.7699 | 0.7840 |
| Ridge regressor (+ sigmoid) | 0.7695 | 0.7833 |
| Convex combination (non-neg, sum=1) | 0.7685 | 0.7822 |
| MLP 元（2 层 16 单元） | 0.7683 | 0.7810 |

L2-LR 胜出，其允许负系数这一特性让 SVM 得到 -0.048 权重起到"去相关"作用；约束凸组合反而损失此自由度。
