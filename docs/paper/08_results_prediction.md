冠军在 5/7 基线上显著更优；对 LR 边界显著（p = 0.069）；与独立调参 RF 差异不显著（p = 0.498）——这正是堆叠理论预期：冠军的 RF 基学习器只是整个堆叠的一部分，在元学习器层面它与 MT-MLP / ET 互补才构成 AUC 提升。**单独拿出 RF 与整体堆叠对比，并不是"谁更强"的问题，而是"哪一个部件在承担主要排序能力"的问题**；SHAP 与消融（Section 7.3–7.4、Section 9）将进一步展开。

## 7.3 SHAP 特征重要性（Table 18，Fig 18）

对权重最大的 RF 基学习器（元系数 0.376）做 Tree-SHAP 得到**模型层**重要性；同时对三个树基学习器（RF + NR-Boost + ET）按归一化的元系数加权聚合得到**堆叠层**重要性。两种方法的 Top-10 排名几乎一致（Spearman rank-correlation $\rho$ = **0.888**）。

**元权重加权 Top-5（堆叠层重要性）**：

1. **工作单位性质** — 重要性 = 0.164
2. **高校类型** — 0.143
3. **收入水平** — 0.130
4. **工作满意度** — 0.112
5. **工作匹配度** — 0.109

此排名中值得注意的两点：

- **"离职意向"未被使用为特征**：本研究训练 / 预测的目标是 `离职行为`（binary），而 `离职意向`（ordinal）是另一个 target，未作为输入。这意味着模型仅凭**客观工作环境 + 工作评价 + 人口学变量**即达到 AUC 0.78，**没有"自报将要离职"这类强信号的污染**。
- **"工作单位性质"与"高校类型"这两项不可干预变量排名最高**：它们是系统性结构变量——例如"外企 + 985 毕业生"的组合与"国企 + 专科毕业生"对离职倾向的结构性差异。HRCF 正因为这类变量不可改变，才把干预资源集中在收入、满意度、匹配度、机会等可干预变量上（Section 8）。

## 7.4 Partial Dependence（Fig 19）

对 Top-5 特征在 RF 上计算 PDP（平均边际效应，30 网格点），展示方向与非线性：

- **收入水平**（越高越稳）：离职概率随收入单调下降，但边际效应在 ¥4k–¥7k 区间最强，¥7k 以上趋于饱和。
- **工作满意度**（Likert 1→5，越高越稳）：几乎线性下降，每升 1 级约降低离职概率 0.03。
- **工作匹配度**（Likert 1→5，越高越稳）：比满意度更陡，尤其从 3 升到 4 时有显著跳变。
- **工作单位性质**（不可干预）：民营/外企 / 国企三类离职率差异约 0.08，模型主要用它做**基线调整**。
- **高校类型**（不可干预）：民办独立学院毕业生离职概率比 985/211 高约 0.10。

**观察**：

- 多数子群 AUC ≥ 0.70，整体稳健。
- 两个子群 AUC < 0.70：**民办/独立高校毕业生**（0.659）和**低收入组**（0.680）。这两组恰好是**正类率最高的子群**（26.3% 和 20.0%），即"最需要干预"的人群同时也是"最难预测的人群"——这是值得在 Limitations 重点报告的发现。
